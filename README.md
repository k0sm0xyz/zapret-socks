# 🥷 zapret-socks

Скрипт автоматической настройки утилиты **Zapret** в режиме изолированного SOCKS5-прокси на порту `1080` для систем на базе Ubuntu (проверено на Ubuntu 26.04 LTS).

## 🚀 Быстрая установка (One-line deploy)

Чтобы развернуть рабочую конфигурацию обхода DPI на новой системе, просто выполните в терминале:

```bash
curl -kLO https://githubusercontent.com && chmod +x install_dpi.sh && ./install_dpi.sh
```

## 🛠️ Что делает этот скрипт:
1. Создает системную директорию `/opt/zapret/binaries/my/`.
2. Автоматически прописывает «золотую» комбо-стратегию (`1,midsld --disorder --oob`) в автозапуск планировщика `cron` при перезагрузке ПК.
3. Полностью вырезает дефолтные конфликтующие системные службы `systemd`, которые могут намертво заблокировать интернет.
4. Поднимает изолированный локальный прокси на порту `1080` для безопасного подключения браузеров (Firefox / Brave).
